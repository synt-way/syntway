**What is `text-emphasis`?**
`text-emphasis` is a CSS property used to apply special emphasis styling to text characters. It allows you to add marks or symbols above, below, or over the text to draw attention to specific parts of the content.

**Why is `text-emphasis` used?**
`text-emphasis` is used to emphasize specific text in a unique way. It can be helpful for highlighting key points, quotations, or other significant content in a visually distinct manner.

**Where is `text-emphasis` applied?**
You can apply the `text-emphasis` property to various HTML elements that contain text content, such as headings, paragraphs, or inline text elements.

**How to use `text-emphasis` with examples:**
Here are some examples of how to use `text-emphasis`:

1. Apply a dot over the text:
   ```css
   .dot-emphasis {
     text-emphasis: dot;
   }
   ```

2. Add a double circle below the text:
   ```css
   .double-circle-emphasis {
     text-emphasis: double-circle;
   }
   ```

3. Emphasize text with a wave above:
   ```css
   .wave-emphasis {
     text-emphasis: wave over;
   }
   ```

4. Apply a filled circle below and above the text:
   ```css
   .filled-circle-emphasis {
     text-emphasis: filled-circle over under;
   }
   ```
