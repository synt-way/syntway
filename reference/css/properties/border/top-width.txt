The `border-top-width` property in CSS is used to set the width of the top border of an element. It allows you to specify how wide the top border should be displayed.

When you want to control the thickness or width of the top border of an element, you use the `border-top-width` property to set that width.

**How to use `border-top-width`?**
The syntax is simple:

```css
element {
  border-top-width: width;
}
```

Replace `width` with the desired width value. This value can be in pixels (px), percentages (%), em, or other length units.

```css
/* Apply a specific width to the top border of an element */
.element {

  border-top-style: solid; /* Define border style */
  border-top-color: blue; /* Define border color */
}
```
