The `border-top-color` property in CSS is used to set the color of the top border of an element. It allows you to specify the color that should be applied to the top border.

**Why is `border-top-color` used?**
When you want to control the color of the top border of an element, you use the `border-top-color` property to define the specific color that should be applied to that border side.

**How to use `border-top-color`?**
The syntax is simple:

```css
element {
  border-top-color: color;
}
```

Replace `color` with the desired color value. This can be a named color, hexadecimal value, RGB value, etc.

Here's an example:

```css
/* Apply a specific color to the top border of an element */
.element {
  border-top-color: blue;
  border-top-width: 2px; /* Define border width */
  border-top-style: solid; /* Define border style */
}
```
