**1. What are Standard Built-In Objects in JavaScript?**
   - **Definition:** Standard built-in objects in JavaScript are a set of objects that are available by default in the JavaScript language. They provide core functionality for common tasks, such as manipulating strings, numbers, dates, arrays, and more.

**2. Why Use Standard Built-In Objects in JavaScript?**
   - **Core Functionality:** These objects offer essential functionality and data types that are fundamental to JavaScript programming.
   - **Efficiency:** They are highly optimized and performant, making them suitable for various tasks.
   - **Consistency:** Standard built-in objects ensure a consistent and predictable programming experience across different JavaScript environments.

**3. Where are Standard Built-In Objects Used in JavaScript?**
   - **JavaScript Code:** They are used directly in JavaScript code to perform tasks like string manipulation, array operations, date handling, and more.

**4. How to Use Standard Built-In Objects:**
   - **Example 1:** Using the `String` object to manipulate strings:
     ```javascript
     var text = "Hello, world!";
     var upperCaseText = text.toUpperCase();
     ```

   - **Example 2:** Using the `Array` object to manipulate arrays:
     ```javascript
     var numbers = [1, 2, 3, 4, 5];
     var doubledNumbers = numbers.map(function(num) {
       return num * 2;
     });
     ```

   - **Date Handling:** The `Date` object is crucial for working with date and time values in JavaScript, including parsing and formatting dates.
   - **JSON:** The `JSON` object is used for working with JSON data, including parsing JSON strings and serializing JavaScript objects into JSON.
