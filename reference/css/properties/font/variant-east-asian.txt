The `font-variant-east-asian` CSS property is used to control the typographic variations specific to East Asian scripts, such as Chinese, Japanese, and Korean. It allows you to apply different stylistic variations and features to East Asian characters.

Here's a detailed explanation of the `font-variant-east-asian` property using the provided template:

### **What is `font-variant-east-asian`?**
The `font-variant-east-asian` property in CSS is used to control the typographic variations specific to East Asian scripts, like Chinese, Japanese, and Korean. It allows you to apply different stylistic alternates, spacing, and other features to East Asian characters.

### **Usage:**
The syntax for using the `font-variant-east-asian` property is as follows:

```css
selector {
  font-variant-east-asian: value;
}
```

Where `selector` is the target element, and `value` represents the desired typographic variations.

### **Example:**
```css
p {
  font-variant-east-asian: ruby;
}

h1 {
  font-variant-east-asian: proportional-width;
}
```

### **Why Use `font-variant-east-asian`?**
The `font-variant-east-asian` property allows you to control and fine-tune the typographic variations of East Asian scripts to achieve desired visual effects and consistency.

### **Where to Use `font-variant-east-asian`?**
Use the `font-variant-east-asian` property on text elements that contain East Asian characters, where you want to apply specific typographic variations.

### **How to Use `font-variant-east-asian`:**
1. Choose an appropriate value for East Asian typographic variations (`normal`, `ruby`, `jis78`, `jis83`, `jis90`, `jis04`, `simplified`, `traditional`, etc.).
2. Apply the `font-variant-east-asian` property to the desired text elements in your CSS.

### **Example Values:**
- `normal`: No East Asian typographic variations are applied.
- `ruby`: Applies ruby (small annotations for pronunciation) typographic style.
- `jis78`, `jis83`, `jis90`, `jis04`: Apply specific Japanese Industrial Standards for characters.
- `simplified`, `traditional`: Switch between simplified and traditional Chinese characters.
