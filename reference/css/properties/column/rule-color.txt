### **What is `column-rule-color`?**
The `column-rule-color` CSS property is used to set the color of the vertical rule (column rule) between columns in a multi-column layout. It specifies the color of the separator line.

### **Usage:**
The syntax for using `column-rule-color` is as follows:

```css
selector {
  column-rule-color: color;
}
```

Where `selector` is the target element, and `color` is the desired color for the column rule. This value can be in various formats, such as color names, hexadecimal codes, RGB values, and more.

### **Example:**
To create a multi-column layout with a green column rule:

```css
.multi-column {
  column-rule-color: green;
}
```

### **Why Use `column-rule-color`?**
Using `column-rule-color` allows you to customize the appearance of the vertical rule between columns. It helps create a visually appealing and organized multi-column layout.

### **Where to Use `column-rule-color`?**
You can use the `column-rule-color` property in multi-column layouts where you want to control the color of the column rule between columns.

### **How to Use `column-rule-color`:**
To create a multi-column layout with a red column rule:

```css
.article {
  column-rule-color: red;
}
```
