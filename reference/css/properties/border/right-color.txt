The `border-right-color` property in CSS is used to set the color of the right border of an element. It allows you to specify the color that should be applied to the right border.

**Why is `border-right-color` used?**
When you want to control the color of the right border of an element, you use the `border-right-color` property to define the specific color that should be applied to that border side.

**How to use `border-right-color`?**
The syntax is simple:

```css
element {
  border-right-color: color;
}
```

Replace `color` with the desired color value. This can be a named color, hexadecimal value, RGB value, etc.

Here's an example:

```css
/* Apply a specific color to the right border of an element */
.element {
  border-right-color: blue;
  border-right-width: 2px; /* Define border width */
  border-right-style: solid; /* Define border style */

```
