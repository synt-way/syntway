### **What is `column-rule-style`?**
The `column-rule-style` CSS property is used to set the style of the vertical rule (column rule) between columns in a multi-column layout. It specifies the appearance of the separator line.

### **Usage:**
The syntax for using `column-rule-style` is as follows:

```css
selector {
  column-rule-style: value;
}
```

Where `selector` is the target element, and `value` specifies the desired style for the column rule. This value can be one of several available styles, such as `solid`, `dashed`, `dotted`, and more.

### **Example:**
To create a multi-column layout with a dashed column rule:

```css
.multi-column {
  column-rule-style: dashed;
}
```

### **Why Use `column-rule-style`?**
Using `column-rule-style` allows you to define the appearance of the vertical rule between columns. It helps enhance the visual organization and aesthetics of your multi-column layout.

### **Where to Use `column-rule-style`?**
You can use the `column-rule-style` property in multi-column layouts where you want to control the style of the column rule between columns.

### **How to Use `column-rule-style`:**
