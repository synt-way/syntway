**What is `border-style`?**
The `border-style` property in CSS is used to set the style of an element's borders. It defines the appearance of the border lines.

**Why is `border-style` used?**
`border-style` is used to control the visual style of an element's borders, whether they appear as solid lines, dashed lines, dotted lines, double lines, and more.

**How to use `border-style`?**
The syntax is simple:

```css
element {
  border-style: style;
}
```

Replace `style` with the desired border style value, such as `solid`, `dashed`, `dotted`, `double`, `none`, etc.

Here's an example:

```css
/* Apply a specific border style to an element */
.element {
  border-style: dashed;
  border-width: 2px; /* Define border width */
  border-color: red; /* Define border color */
}
```

**Where to use `border-style`?**
This property is used within your CSS styles and can be applied to various HTML elements that have borders, such as divs, paragraphs, and more.
