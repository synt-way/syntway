**What is `border-top-style`?**
The `border-top-style` property in CSS is used to set the style of the top border of an element. It allows you to specify how the top border should be displayed, whether it's solid, dashed, dotted, double, or another available style.

**Why is `border-top-style` used?**
When you want to control the visual style of the top border of an element, you use the `border-top-style` property to define how that border should be rendered.

**How to use `border-top-style`?**
The syntax is simple:

```css
element {
  border-top-style: style;
}
```

Replace `style` with the desired border style value, such as `solid`, `dashed`, `dotted`, `double`, and others.

Here's an example:

```css
/* Apply a specific border style to the top side of an element */
.element {
  border-top-style: dashed;
  border-top-width: 2px; /* Define border width */
  border-top-color: red; /* Define border color */
}
```

This property is used within your CSS styles and can be applied whenever you need to customize the style of the top border of an element.
