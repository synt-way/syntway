### **What is `column-gap`?**
The `column-gap` CSS property is used to set the size of the gap between columns in a multi-column layout. It defines the spacing between adjacent columns.

### **Usage:**
The syntax for using `column-gap` is as follows:

```css
selector {
  column-gap: value;
}
```

Where `selector` is the target element, and `value` specifies the desired size of the gap between columns. This value can be in various units, such as pixels (`px`), ems (`em`), percentages (%), etc.

### **Example:**
To create a multi-column layout with a gap of 20 pixels between columns:

```css
.multi-column {
  column-gap: 20px;
}
```

### **Why Use `column-gap`?**
Using `column-gap` helps control the spacing between columns in a multi-column layout. It's useful for creating visually appealing layouts and improving readability by preventing content from feeling too crowded.

### **Where to Use `column-gap`?**
You can use the `column-gap` property in multi-column layouts, particularly when you want to set a consistent gap between columns for better separation.

### **How to Use `column-gap`:**
To add a gap of 10% between columns in a multi-column layout:

```css
.content {
  column-gap: 10%;
}
```
