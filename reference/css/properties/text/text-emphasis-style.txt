**What is `text-emphasis-style`?**
`text-emphasis-style` is a CSS property used to specify the style (the appearance or pattern) of emphasis marks created by the `text-emphasis` property. Emphasis marks are decorative symbols or characters that can be added above, below, or over text to draw attention to specific parts of the content.

**Why is `text-emphasis-style` used?**
`text-emphasis-style` is used to control the visual style or pattern of emphasis marks applied to text. It allows you to customize the appearance of these marks, creating unique visual effects to enhance the presentation of text.

**Where is `text-emphasis-style` applied?**
You can apply the `text-emphasis-style` property to various HTML elements that contain text content, such as headings, paragraphs, or inline text elements.

**How to use `text-emphasis-style` with examples:**
Here are some examples of how to use `text-emphasis-style`:

1. Apply a solid underline as the emphasis style:
   ```css
   .solid-underline-emphasis {
     text-emphasis: underline;
     text-emphasis-style: solid;
   }
   ```

2. Create a dotted overline for emphasis:
   ```css
   .dotted-overline-emphasis {
     text-emphasis: overline;
     text-emphasis-style: dotted;
   }
   ```

3. Use a wavy line as the emphasis style:
   ```css
   .wavy-emphasis {
     text-emphasis: dots;
     text-emphasis-style: wavy;
   }
   ```
