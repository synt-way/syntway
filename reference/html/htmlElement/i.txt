**1. What is the `<i>` element?**
The `<i>` element in HTML is used to apply italic formatting to text within the content. It has a presentational meaning and is used to indicate that the enclosed text should be displayed in italic font style. However, it doesn't carry any semantic meaning on its own.

**2. Why use the `<i>` element?**
The `<i>` element is used to visually emphasize or highlight specific words or phrases by rendering them in italic font style.

**3. Where can the `<i>` element be used?**
The `<i>` element can be used within various parts of your HTML content where you want to apply italic formatting to specific words or phrases.

**4. How to use the `<i>` element?**
Wrap the text you want to italicize within the `<i>` element. Here's an example:
```html
<p>This is a <i>highlighted</i> word.</p>
```
